# ref-fr-STIF
Des scripts pour le suivi de l'intégration des référentiels île-de-France Mobilités (ex STIF) dans OSM.

## Prérequis
* [xsv](https://github.com/BurntSushi/xsv)
* osmium-tool
* python3 et poetry
